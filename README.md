# Integrated Database Project

An integrated PostgreSQL database project that combines social media, payment system, and career platform structures into a single database architecture.

## Technologies Used
- PostgreSQL
- SQL
- ER/EER Database Modeling

## Project Features
- Relational database schema design
- CRUD operations
- Nested queries
- JOIN and aggregate queries
- Views
- Triggers
- Stored Procedures

## Project Structure
- `schema.sql` → Database schema creation
- `sample_data.sql` → Sample data insertion
- `select_queries.sql` → Basic SELECT queries
- `nested_select_queries.sql` → Nested SQL queries
- `joins_and_aggregates.sql` → JOIN and aggregate operations
- `update_queries.sql` → UPDATE operations
- `delete_queries.sql` → DELETE operations
- `views.sql` → SQL views
- `triggers.sql` → Database triggers
- `stored_procedures.sql` → Stored procedures
